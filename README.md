### To Run Frontend Locally

1. ##### Clone the project
    * git clone `https://github.com/aniketgarole/English_Quest_Assignment.git`

2. ##### Go to project directory
    * cd frontend

3. ##### Install dependencies
    * npm install

4. ##### Start the server
 * npm run start




### To Run Backend Locally

1. ##### Clone the project
    * git clone `https://github.com/aniketgarole/English_Quest_Assignment.git`

2. ##### Go to project directory
    * cd backend

3. ##### Install dependencies
    * npm install

4. ##### Start the server
    * npm run start
